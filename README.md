## Description
This Json version is an array of object of the following shape:
- Name
- Complete Address
- URL
- Latitude
- Longitude
- Day cost for a desk
- Monthly cost for a flex desk
- Monthly cost for a fixed desk
- Notes

```
[
  {
      "name": "",
      "address": "",
      "url": "",
      "lat": "",
      "lng": "",
      "cost_day": "",
      "cost_flex": "",
      "cost_fixed": "",
      "notes": ""
  },
  ...
]
```
